Crystal Collector Game (Pascal)

Instructions

The game will generate a random number at the start of the game in the 'Number to match' field.

There are four colored crystals below (red, blue, green, yellow). By clicking on any crystal, you will add a specific amount of points to your total score.

The point value of each crystal is hidden until you click on it. After clicking on each crystal, you will see the 'Your number' incremently increase by the number of points hidden behind each crystal.

You win the game by matching your total score to the random number. You lose the game if your total score goes above the random number.

Each time when the game starts (or starts over), the game will change the values of each crystal.
